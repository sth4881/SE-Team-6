# SE-Team-6

### ‘내가 가진 식재료를 바탕으로 메뉴를 추천해주는 어플’
- 내가 보유한 식재료를 토대로 곧바로 조리할 수 있는 음식들을 보여줌으로써 어떤 요리를 만들어서 먹을지 고민하지 않아도 된다.
- 본인이 가지고 있는 식재료에서 음식 조리에 사용된 재료를 자동으로 차감함으로써 식재료가 얼마나 남았고, 장을 언제 보러 가야 할지 파악할 수 있다.
